Lëson 1: Sartu Lire bu mat sekk

Mbooleem bindeen yi ak ni ñuy waxee baat yu nekk ci Faransé

1. Mbooleem Araf yi (A ba Z)

Araf par araf, ni ñuy waxee turam ci Faransé :

A -> (a)
B -> (bé)
C -> (sé)
D -> (dé)
E -> (e)
F -> (ef)
G -> (jé)
H -> (ac)
I -> (i)
J -> (ji)
K -> (ka)
L -> (el)
M -> (em)
N -> (en)
O -> (o)
P -> (pé)
Q -> (kü)
R -> (er)
S -> (es)
T -> (té)
U -> (ü)
V -> (vé)
W -> (dublé-vé)
X -> (iks)
Y -> (igrek)
Z -> (zed)

2. Sartu boole araf yi (Syllabes)

Jàngal ni ngay boolee consonne ak voyelle ngir rëcc kàddu yi :

B + A = BA -> (ba) | B + E = BE -> (be) | B + I = BI -> (bi)
B + O = BO -> (bo) | B + U = BU -> (bü)
M + A = MA -> (ma) | M + E = ME -> (me) | M + I = MI -> (mi)
M + O = MO -> (mo) | M + U = MU -> (mü)
Misal bu yomb: Maman (ma man) -> Firi: Yaay
Misal bu yomb: Papa (pa pa) -> Firi: Baay

3. Voyelles Nasales (Yi ñuy waxee ci bakkan)

Yii dañu dakk ci bakkan, daway soppi waxiinu araf yi :

AN / AM / EN / EM
Léral waxiinu: Melni "an" ci Wolof bu jaxasoo ak "on"
Misal: Enfant (an fan) -> Firi: Xale
IN / IM / AIN / EIN
Léral waxiinu: Melni "en" bu dakk ci bakkan
Misal: Pain (pen) -> Firi: Mburu

4. Bindeen yi gëna jafee lire

Booleem araf yii buy jaxasoo di am beneen waxiin wu bees :

CH
Léral waxiinu: Day nekk "C" (mélni "sh" ci angale)
Misal: Chat (ca) -> Firi: Muus
OI
Léral waxiinu: Day nekk "wa" tàllal
Misal: Oiseau (wazo) -> Firi: Picc

5. Baat yu niroo ci Wolof ak Faransé

Français: Chaussettes -> (ci Wolof: soté) -> Firi: Soté
